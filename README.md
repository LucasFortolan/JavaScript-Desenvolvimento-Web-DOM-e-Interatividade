# Estudos de JavaScript

Este repositório foi criado para registrar meus estudos e práticas em **JavaScript**, com base em aulas e materiais do **Curso em Vídeo (Gustavo Guanabara)**, além de exercícios e desafios aplicados.

O objetivo principal foi construir uma base sólida em programação para desenvolvimento web, entendendo como o JavaScript funciona no navegador e como ele pode ser utilizado para criar páginas dinâmicas e interativas.

## Objetivo do Projeto

- Aprender os fundamentos da linguagem JavaScript
- Desenvolver lógica de programação aplicada ao front-end
- Criar interações com o usuário e com elementos HTML através do DOM
- Consolidar conhecimentos por meio de exercícios e desafios práticos

## Conteúdos Aprendidos

Durante os estudos, foram abordados temas como:

- Estrutura básica do JavaScript e funcionamento no navegador
- Interações com o usuário (`alert`, `prompt`, `confirm`)
- Manipulação de elementos da página usando **DOM**
- Variáveis (`var`, `let`) e tipos de dados
- Conversão de dados (String → Number / Number → String)
- Operadores aritméticos, relacionais e lógicos
- Condicionais (`if`, `else`, `switch`, operador ternário)
- Funções e reutilização de código
- Arrays e métodos como `splice()`, `findIndex()` e `forEach()`
- Objetos e manipulação de propriedades
- Eventos em JavaScript (cliques, mouseover, etc.)
- Referenciamento de arquivos externos
- Inserção e alteração dinâmica de imagens via JavaScript
